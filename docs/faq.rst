Frequently Asked Questions
==========================

General Questions
-----------------

**What is StudyFlow?**

StudyFlow is an open-source task and project management tool designed specifically for university 
students and academic teams. It helps you organize courses, track deadlines, and collaborate on 
group projects.

**Is StudyFlow free?**

Yes! StudyFlow is completely free and open-source under the Apache 2.0 License. You can use it, 
modify it, and even self-host it without any cost.

**What makes StudyFlow different from other task managers?**

Unlike generic task managers, StudyFlow is built specifically for academic workflows. It 
understands the unique needs of students managing multiple courses, group projects, and deadlines.

Technical Questions
-------------------

**What technologies does StudyFlow use?**

Currently, the project includes:

* Jekyll for the marketing website
* Sphinx for documentation
* Future application will use JavaScript/Node.js

**Can I self-host StudyFlow?**

Yes! One of our core principles is privacy-first. You can self-host StudyFlow to keep your 
academic data completely private.

**What are the system requirements?**

For the Jekyll website:

* Ruby 2.7+
* Bundler

For future application:

* Node.js 16+
* Modern web browser

**Does StudyFlow work offline?**

Offline support is planned for future releases.

Getting Started
---------------

**How do I install StudyFlow?**

See our :doc:`getting_started` guide for detailed installation instructions.

**I'm getting errors when running Jekyll. What should I do?**

Make sure you have:

1. Ruby 2.7 or higher installed
2. Bundler installed (``gem install bundler``)
3. Run ``bundle install`` in the website directory
4. Try ``bundle exec jekyll serve`` instead of just ``jekyll serve``

We welcome contributions! Check out our :doc:`contributing` guide and the `CONTRIBUTING.md

Please open an issue on our `GitHub Issues page

Absolutely! We love new ideas. Open an issue on GitHub or join our `Discord server

Not at all! You can contribute by:

* Reporting bugs

* Improving documentation
* Helping other users in the community
* Sharing StudyFlow with fellow students

Privacy & Data
--------------

**What data does StudyFlow collect?**

As a self-hosted, privacy-first application, StudyFlow doesn't collect any personal data. Your 
information stays on your own server.

**Is my study data secure?**

When self-hosted, you have complete control over your data security. We recommend following 
standard security best practices for your hosting environment.
